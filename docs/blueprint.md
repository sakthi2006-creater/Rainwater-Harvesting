# **App Name**: AquaVest

## Core Features:

- Input Parameters: Collects roof area, location (via map selection), and population data from the user.
- Rainfall Data Fetch: Retrieves rainfall and groundwater data based on the selected location, using an available API or preloaded dataset. This can be simulated with dummy rainfall data for MVP.
- Harvesting Potential Calculation: Calculates rainwater harvesting potential using the formula: Runoff = Rainfall × Area × Coefficient (0.85).
- Suggested Structure: Suggests the appropriate size for a water storage tank based on the calculated runoff potential.
- Clean Dashboard Output: Presents the harvesting potential, suggested structure, and cost-benefit analysis in a user-friendly dashboard.
- IoT Rainfall Simulation: Simulates real-time rainfall data using generated random rainfall values to show rainfall as if sensed by a sensor.
- Report Generation: Offers a downloadable report (PDF format) containing calculated water harvesting potential and suggested storage solution.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to represent water and reliability.
- Background color: Light blue (#E3F2FD) to create a clean and refreshing interface.
- Accent color: Green (#4CAF50) to highlight calls to action and represent sustainability.
- Body and headline font: 'PT Sans', a humanist sans-serif that balances a modern look with approachability, suitable for both headlines and body text.
- Use clear, modern icons to represent water, structures, and data points. A visual language featuring line-art icons is preferrable.
- Prioritize a clean, intuitive layout with a clear hierarchy of information, making data easy to digest and interact with.
- Use subtle animations to enhance user experience, such as smooth transitions between data views.
